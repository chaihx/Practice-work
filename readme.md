# 项目背景

软件工程课程实践作业四《电子族谱系统》，族谱又称家谱、宗谱，记录着一个家族的来源与变迁，是一个家族的血脉史，族谱文化是中华传统优秀文化的重要组成部分。电子族谱与传统纸质族谱相比，有其独特优势。本项目围绕电子族谱系统展开。
